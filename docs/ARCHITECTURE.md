# 🛰️ Architecture & Visual Identity: Silicon Commodity

## 1. Core Concept
The site is designed as an **Industrial Terminal / Command Center**. It prioritizes information density, technical precision, and a "living system" aesthetic.

## 2. Technical Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Typography**: 
  - Sans-serif: UI & Headings
  - Monospace: Data, Logs, IDs, Metadata
- **Rendering**: Custom dual-mode engine (Markdown + Dangerous HTML Injection)

## 3. Visual Standards
- **Container**: Global `max-w-7xl` with `px-8` padding.
- **Alignment**: Strict **Left Alignment**. Header, Navigation, Hero, and Content must all align to the same left boundary.
- **Colors**:
  - Background: `slate-950` (#020617)
  - Primary Accent: `cyan-500` (#06b6d4)
  - Borders/Lines: `white/10` or `white/5`
- **Atmosphere**: Professional, Cold-start, Algorithm-driven.
